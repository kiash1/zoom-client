# Zoom client
* Create zoom meetings 
* Powered by [fast-api](https://fastapi.tiangolo.com/)


## Usage(local):
* Install docker and docker-compose
* Run `docker-compose  up --build -d`
* Use api's as intended
* Change values in env as you see fit
